本网站介绍了**56种中国珍稀野生动物**，涵盖哺乳动物、鸟类、爬行动物、两栖动物、鱼类、昆虫与无脊椎动物六大类，全部配有精美的儿童插画风格图片。

适合**小学生、自然爱好者**在线阅读学习。

## 🚀 部署方法

### 方式一：GitHub Pages（推荐）

1. 在 GitHub 创建仓库，命名为 `中国野生动物百科`
2. 将本文件夹所有文件推送上去
3. 在仓库 Settings → Pages 中启用
4. 选择 `main` 分支，根目录
5. 等待几分钟即可通过 `https://用户名.github.io/中国野生动物百科/` 访问

### 方式二：其他静态托管

将整个文件夹拖入 Netlify / Vercel / Cloudflare Pages 即可自动部署。
